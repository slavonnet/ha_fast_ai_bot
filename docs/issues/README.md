# Issue Seed Pack

Пакет подготовленных issues для горизонтальной декомпозиции проекта.

## Как использовать
1. Создать Milestones `Stage 0` .. `Stage 9`.
2. Создать Meta issues из `docs/issues/meta/*.md`.
3. Создать Subtask issues из `docs/issues/subtasks/*.md` и проставить parent/dependencies.
4. Для найденного техдолга создавать отдельные issues по шаблону `Tech Debt`.

## Горизонтальные потоки (Tracks)
- H1 Contracts & Schemas
- H2 Runtime & Execution
- H3 Integration & Reliability
- H4 QA, Docs & Release
